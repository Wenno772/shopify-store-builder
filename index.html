Comment installer une section dans Shopify
  1. Va dans Boutique en ligne → Thèmes
  2. Clique sur Personnaliser sur le thème Dawn
  3. En bas à gauche, clique Ajouter une section
  4. Choisis HTML personnalisé
  5. Colle le code de la section dans le champ HTML
  6. Clique Enregistrer — c'est en ligne !
Installer le thème Dawn (gratuit)
  1. Dans Shopify : Boutique en ligne → Thèmes
  2. Clique Visiter la thème store
  3. Recherche "Dawn" → Ajouter
  4. Publie Dawn comme thème principal
